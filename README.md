# 🚢 Titanic Survival Prediction

A Machine Learning project that predicts whether a passenger survived the Titanic disaster using the famous Kaggle Titanic dataset. This project demonstrates the complete machine learning workflow, from data preprocessing and feature engineering to model training and evaluation using multiple classification algorithms.

---

## 📌 Project Overview

The objective of this project is to build predictive models that determine passenger survival based on features such as age, gender, passenger class, fare, embarkation point, and cabin information.

The project focuses on:

- Data exploration
- Handling missing values
- Feature engineering
- Data preprocessing
- Model training
- Performance evaluation
- Comparison of multiple machine learning algorithms

---

## 📊 Dataset

The project uses the **Titanic - Machine Learning from Disaster** dataset from Kaggle.

Dataset features include:

- PassengerId
- Pclass
- Name
- Sex
- Age
- SibSp
- Parch
- Ticket
- Fare
- Cabin
- Embarked

**Target Variable**

- Survived

---

## 🛠 Data Preprocessing

The notebook performs several preprocessing steps including:

- Handling missing values
- Cleaning categorical attributes
- Feature engineering
- Encoding categorical variables
- Preparing data for machine learning models

## 🤖 Machine Learning Models

The project compares several classification algorithms, including:

- Decision Tree Classifier
- Random Forest Classifier
- Support Vector Machine (SVM)
- XGBoost Classifier

Each model is evaluated to compare prediction performance.

The models are evaluated using common machine learning metrics such as:

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/yourusername/titanic-survival-prediction.git

cd titanic-survival-prediction
```

### Install dependencies

```bash
pip install pandas
pip install numpy
pip install matplotlib
pip install seaborn
pip install scikit-learn
pip install xgboost
pip install jupyter
```

Or install everything together:

```bash
pip install pandas numpy matplotlib seaborn scikit-learn xgboost notebook
```

### Run the notebook

```bash
jupyter notebook
```

Open:

